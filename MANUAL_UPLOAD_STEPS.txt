MANUAL STEPS TO UPLOAD TO GITHUB
================================

If the batch files or PowerShell scripts aren't working, follow these manual steps:

1. Open Command Prompt (cmd.exe)

2. Navigate to the repository directory:
   cd C:\Users\PC 01\OneDrive\Desktop\TasteMakers\pagodnaako

3. Add all files to git:
   git add .

4. Commit the changes:
   git commit -m "Update TasteMakers project"

5. Push to GitHub:
   git push origin main

6. If prompted for credentials, enter your GitHub username and password or personal access token.
   (You may need to create a personal access token at https://github.com/settings/tokens)

That's it! Your files should now be uploaded to GitHub at:
https://github.com/hatdogshanghai/pagodnaako.git
